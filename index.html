blio
It's an acronym for "flat array research system". 

          

          

            
            

          

        


        

          A flat APL in King Nested's court. (Hmm, that's rather backwards isn't it...)
          

            Blio is a modern APL which utilizes a fully-flat model (no nesting, no boxing!).  It provides features that alleviate the pains of the flat model and incorporate newer thinking in array programming.

            Blio is designed for hardware applications.
